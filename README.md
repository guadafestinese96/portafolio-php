# Portfolio en PHP

Portfolio personal desarrollado con PHP y MySQL, ideal para hosting en servidores tradicionales.

## ⚙️ Tecnologías
- PHP
- MySQL
- HTML
- CSS
- JavaScript

## 🧩 Funcionalidades
- Presentación personal
- Listado de proyectos
- Formulario de contacto funcional con PHP
- Panel de administración para editar contenidos (si aplica)

## 💻 Instalación
1. Clonar el repositorio
2. Subir a un servidor con PHP (ej: XAMPP, InfinityFree, etc.)
3. Crear la base de datos con el archivo `.sql` incluido (si hay)
4. Configurar conexión en `config.php` o archivo similar

## 📂 Archivos importantes
- `index.php`: página principal
- `config/`: variables de entorno y conexión
- `admin/`: zona de administración (opcional)
